# WorldGenerator
 A basic 2D World Generator I wrote out of spite, due to every other World Generator and Game out there using Perlin Noise.

# Controls
| Key | Function                         |
| --- | -------------------------------- |
| Q   | Exit                             |
| W   | Export as BMP                    |
| E   | Re-render map                    |
| R   | Render new map                   |
| T   | Rotate sun (laggy)               | 
| A   | Save Albedo as BMP               |
| S   | Save Heightmap as BMP            |
